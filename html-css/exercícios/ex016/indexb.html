Graphic Design is my Passion

Para definir configurações globais do CSS utilizamos: *{}.

Para criar um efeito de gradiente no fundo do site, utilizamos o parâmetro 'background-image' com o valor 'linear-gradient', seguido pelas cores que desejamos inserir no "degradê". Dentro do 'linear-gradient' podemos inserir atributos que definem como o gradiente se comporta (esquerda para direita, baixo para cima, angulação, etc). Além disso, existem outros tipos de gradientes como o 'angular-gradient'.

Para não criar um site como esse (gradiente do instagram), e utilizar cores mais harmônicas, algumas ferramentas interessantes são: adobecolor, paletton e coolors.co.
