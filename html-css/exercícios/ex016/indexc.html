Paleta Monocromática

Esse é um exemplo de site com uso de uma paleta de cores monocromática.

Novos Estilos

Nesse site foram utilizadas os novos estilos em CSS: border-radius (arredondar as bordas), box-shadow (adicionar sombreamento), padding (margem à direita) e margin (posicionamento do conteúdo).
