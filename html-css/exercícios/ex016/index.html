Representação das Cores

Por nomes

Em HTML é possível representar as cores simplesmente através dos seus nomes:

Roxo

Por código hexadecimal

A representação por código hexadecimal funciona através de 6 dígitos que podem assumir valores de 0 (menos significativo) à F (mais significativo). Cada par de dígitos representam a intensidade de uma da cores primárias (Azul, Vermelho e Verde).

Laranja

Por código RGB

O código RGB funciona de maneira semelhante ao código hexadecimal, porém, atribuindo um valor de 0 a 255 para cada cor primária.

Verde

Por HSL

A representação por HSL define as cores através dos parâmetros Matriz (Hue), Saturação (Saturation) e Luminosidade (Luminosity).

Azul

Opacidade ou Transparência

Todas as formas de representação acima, além de serem capazes de representar as cores, também são capazes de representar a transparência, através do parâmetro Alpha.
