Quais são as principais formatações de texto em html?

Negrito ou Destaque

Para deixar um termo em negrito, devemos usar a tag <b>. Entretanto, essa tag possui apenas forma, e não significado.

Para destacar um termo usando semântica, devemos utilizar a tag <strong>,

Itálico

De forma semelhante, para alterar apenas a forma de um termo, deixando ele em itálico, devemos usar a tag <i>.

Já para adicionar significado e deixar um termo em enfase, devemos utilizar a tag <em>.

Big e Small

É possível alterar o tamanho da fonte utilizando as tags <big> e <small>. Entretanto, diferente da tag small, a tag big é considerada obsoleta.

Texto inserido

Para indicar que um texto foi adicionado posteriormente, podemos utilizar a tag <ins>.

Texto excluído

Para indicar que um texto deve ser desconsiderado, utilizamos a tag <del>.

Texto sobrescrito

Para escrever um texto sobrescrito, como é o caso de equações com potenciação, devemos utilizar a tag <sup>: x2+7.

Texto subscrito

Já para escrever um texto subscrito, como em fórmulas químicas, utilizamos a tag <sub>: H2O
