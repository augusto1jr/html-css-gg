Semântica

Em html5 algumas tags se tornaram obsoletas por trazeram apenas uma mudança de forma, como é o caso das tags: <font>, <center> e <bgcolor>.

Em contrapartida, surgiram novas tags que buscam trazer semântica, ou seja, significado para algumas partes do conteúdo do site, como é o caso das tags: <strong>, <em>, <del>, <mark>, <ins> e

<adress>
.

Mudar a cor da marcação da tag <mark> não altera o seu significado, portanto, deve ser feito por CSS e não por HTML.
