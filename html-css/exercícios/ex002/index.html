Como funcionam os parágrafos em html?

Os paragráfos devem ser inseridos dentro das tags <p>

A tag <hr> pode ser usada para adicionar uma linha

A tag <br> pode ser usada para quebrar uma linha


Para inserir os caracteres das tag (< >) deve-se usar os comandos: "&lessthan" e "&greaterthan".
