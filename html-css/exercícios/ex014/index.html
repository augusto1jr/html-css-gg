Estilos Locais ou Internos

Os estilos internos funcionam através do uso da tag <style> dentro do cabeçalho (<head>) do arquivo .html.

Dessa forma, os parâmetros de estilos são declarados apenas uma vez, e aplicados em todo o corpo do site.

Entretanto, para sistemas que contenham diversos arquivos .html, ainda sim a manutenção se torna alta quando é necessário alterar o estilo de todos os arquivos.
