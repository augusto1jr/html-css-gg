Como utilizar imagens em html

É importante que as imagens utilizadas sejam livres de copyright ©.

Para isso, boas opções de site com imagens gratuitas para uso são: unsplash, pexels e freepik.

Também é importante utilizar imagens compactadas, pois mesmo que isso sacrifique um pouco da qualidade da imagem, é fundamental para manter o seu site "leve". Para isso, uma boa opção de software gratuito é o gimp.

O melhor formato de compactação é o .jpeg, mas o .png permite a utilização de imagens com fundo transparente.

É possível utilizar imagens armazenadas em subpastas, outros caminhos locais e até mesmo armazenadas na internet.
