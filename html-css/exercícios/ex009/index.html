Listas ordenadas

Para criar listas ordenadas em HTML utilizamos a tag <ol> para limitar todos os itens da lista, e a tag <li> para identificar cada um deles.

A tag <ol> possui um parâmetro "type" onde configuramos o tipo de marcador da lista, e um parâmetro "start" onde definimos de onde começa a contagem.

  1. type=1 - Valor padrão: 1,2,3,4,5...
  2. type=A - Letras alfabéticas maiúsculas: A,B,C,D,E...
  3. type=a - Letras alfabéticas minúsculas: a,b,c,d,e...
  4. type=I - Algarismos romanos maiúsculos: I,II,III,IV,V...
  5. type=i - Algarismos romanos minúsculos: i,ii,iii,iv,v...

Listas não ordenadas

Para as listas não ordenadas, utilizamos a tag <ul>.

De forma semelhante, a tag <ul> também possui um parâmetro "type" para definir o tipo de marcador da lista.

  • disc = círculo preenchido
  • circle = círculo vazado
  • square = quadrado

Listas de Definições

Para as listas de definições utilizamos a tag <dl>. Essa é uma lista sem demarcação, e é utilizada para atribuir uma definição <dd> à um termo <dt>.
