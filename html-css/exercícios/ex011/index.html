Imagens Dinâmicas

Imagens dinâmicas em HTML são imagens cujo conteúdo ou atributos (como a fonte da imagem, tamanho ou estilo) podem ser alterados dinamicamente durante a execução de uma página web, sem a necessidade de recarregá-la.

Source

Utilizamos a tag <source> (source media type) para definir a largura limite, que se ultrapassado, irá substituir a imagem por outra que seja compatível com o tamanho do navegador.

Áudio

Para reproduzir um áudio em HTML5 devemos utilizar a tag <audio> em conjunto com os atributos "controls", "loop", "preload" e "autoplay".

Vídeo

Vídeos em hospedagem própria

Quando o vídeo está hospedado no próprio servidor.

Para reproduzir um vídeo em HTML5 devemos utilizar a tag <video> em conjunto com os atributos "controls", "loop", "poster" e "autoplay".

Assim como nas imagens, é importante se atentar para o trade-off entre qualidade do vídeo e velocidade de carregamento do site. Um software interessante para redimensionar ou converter os formatos de vídeo é o HandBrake.

Outro ponto importante é se atentar em disponibilizar os áudios e vídeos em mais de um formato, para oferecer compatibilidade com uma maior gama de navegadores.

Vídeos hospedados externamentes

Hospedar um vídeo internamente pode aumentar os custos de hospedagem. Para evitar esse problema, uma possível solução é utilizar uma hospedagem externa, como o YouTube ou o Vimeo.
