Estilos Externos

Os estilos externos funcionam através de um arquivo .css que contém todos os parâmetros de estilos, sendo externos ao arquivo .html.

Dessa forma, os parâmetros de estilos são declarados apenas uma vez no arquivo .css, e aplicados em todos os arquivos .html do sistema.

Portanto, devemos utilizar o css externo para configurações de estilos gerais, e somente quando necessário incorporar o css interno ou inline para estilos mais específicos.
