Efeito Zebrado

Para tabelas maiores, podemos utilizar o efeito zebrado para melhorar a visibilidade. Fazemos isso através da pseudoclasse "nth":

table tbody tr:nth-child(2n)

Podemos utilizar como parametro o número de colunas (2n, 3n, 4n...) ou os termos ímpar (odd) e par (even).

Também pensando em tabelas maiores, podemos utilizar o parametro "sticky" com valor "-1" para manter o cabeçalho fixo no topo ao rolar a tela.
