iFrame

É possível utilizar os iFrames para exibir conteúdo de outros sites internamente. O tamanho dos iFrames podem ser personalizados via inline ou css. É uma boa prática disponibilizar o link do site para navegadores não compatíveis com iFrame.

  • Projeto Poema
  • Projeto Tech English

Através do >srcdoc<, é possível determinar um conteúdo estático a ser pré-exebido no iFrame antes da primeira interação do usuário.

Também é importante garantir a segurança dos iFrames, através dos parâmetros sandbox e referer-policy, que evitam que os dados inseridos no iFrame sejam capturados.
