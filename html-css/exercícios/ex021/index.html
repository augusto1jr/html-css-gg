Modelo de Caixas

  • height: altura
  • width: largura
  • border: borda formada pelas linhas que circundam o conteúdo
  • padding: espaçamento entre o conteúdo e a borda
  • margin: espaçamento externo à borda
  • outline: traçado da borda
  • box-shadow: sombreamento de um elemento
  • box-level: quebra a linha e isola o conteúdo a ser exibido (<div>);
  • inline-level: exibe o conteúdo dentro da linha (<span>).

Para usar shorthands ao configurar o border ou outline, seguimos a ordem: width -> style -> color

Já para configurar o padding ou margin com shorthands, utilizamos o sentido horário: top -> right -> bottom -> left

Para configurar o sombreamento também utilizando shorthands, utilizamos a sequência: distanciamento horizontal -> distanciamento vertical -> desfoque -> espalhamento
