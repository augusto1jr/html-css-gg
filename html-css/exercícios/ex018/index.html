Fonte Externa

É possível importar fontes externas por meio de links através do '@import'. Para obter esses links podemos utilizar o Google Fonts.

Também é possível utilizar fontes baixadas externamente através do '@font-face' referenciando o arquivo (.otf, .ttf, .aat, .svg). Para encontrar essas fontes, uma boa recomendação de site é o dafont.

Como encontrar fontes?

Além do uso de sites como o Google Fonts e o DaFont, outras ferramentas bem-vindas para identificar outras fontes usadas na internet são o Fonts-Ninja (em texto) e WhatFontIs, FontSquirrel e MyFonts (em imagens).
